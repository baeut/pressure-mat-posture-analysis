# 📦 坐姿识别压力垫 — 产品规格

> 产品化规范书 v1.0

## 产品定位
智能坐垫 — 集**坐姿监测、久坐提醒、座椅加热通风**于一体的健康办公配件。
目标用户：办公室白领、程序员、游戏玩家、康复人群。

---

## 硬件规格

| 项目 | 参数 |
|:-----|:------|
| 尺寸 | 45×45×3cm (坐垫) + 8×5×3cm (控制盒) |
| 重量 | ~800g |
| 主控 | ATmega328P-AU (TQFP-32) |
| 传感器 | 6× FSR 薄膜压力传感器 |
| 温湿度 | DHT22 (±0.5°C, ±2%RH) |
| 加热 | 碳纤维 12V 15W，PWM可调 |
| 通风 | 静音风扇 12V 0.1A，PWM可调 |
| 供电 | 12V/2A DC 适配器 |
| 通信 | USB 串口 / 蓝牙 HC-05 (选配) |
| 接口 | USB Type-C |

## 指示灯说明

| 颜色 | 含义 |
|:-----|:------|
| 蓝灯慢闪 | 系统运行正常，无人 |
| 绿灯常亮 | 有人，坐姿端正 |
| 黄灯常亮 | 有人，坐姿不良 |
| 红灯闪烁 (3s) | 久坐提醒 |
| 红灯快闪 (1s) | 低电量/故障 |

## 安全保护

| 风险 | 缓解措施 |
|:-----|:---------|
| 加热过热 | 30分钟自动关闭 + 65°C 温度保险丝 |
| 风扇堵转 | PWM 电流检测 + 自动关闭 |
| 电池过放 | 3.0V 低电自动关机 |
| 传感器失效 | 自检失败→串口报错→LED红灯 |
| 软件死机 | 硬件看门狗 (WDT) 8秒复位 |
