## 固件架构

### 核心循环

```
loop()
 ├── read_sensors()        — 6路ADC采集，每路20次取中值
 ├── normalize()           — 减去空载基准，映射到0~100%
 ├── calc_features()       — 计算压力重心(COG)、左右/前后对称比
 ├── classify_posture()    — 阈值逻辑判断坐姿
 ├── check_sit_time()      — 累计坐姿时间，触发久坐提醒
 ├── format_json()         — 组装 JSON 数据包
 └── serial_send()         — 串口输出
```

### 主要数据结构
```cpp
// 6路传感器原始值
int sensorRaw[6];

// 归一化压力百分比 (0~100)
float sensorNorm[6];

// 坐姿类型枚举
  Posture = {
    NOBODY = 0,
    UPRIGHT = 1,
    LEAN_FORWARD = 2,
    LEAN_BACK = 3,
    LEAN_LEFT = 4,
    LEAN_RIGHT = 5,
    CROSS_LEFT = 6,
    CROSS_RIGHT = 7,
    OBJECT = 8     // v1.3新增：物品（无压力波动）
  };

// 输出的 JSON 串
// {"t":12345,"s":[12.3,45.6,...],"p":1,"pn":"端正坐姿","sit_m":30}
```

### 阈值配置（`ml_model.h`）
```cpp
// 空载阈值（传感器读数低于此值视为无人）
#define THRESHOLD_EMPTY    10

// 坐骨最小压力（确认有人坐着）
#define THRESHOLD_OCCUPIED 30

// 对称比阈值（判断左右倾斜）
#define THRESHOLD_SYMM     0.25

// 二郎腿检测阈值
#define THRESHOLD_LEG      0.7

// 前后比阈值（判断前倾后仰）
#define THRESHOLD_TILT     0.4
```

## v1.3 新增：物品检测

> 基于**压力波动方差分析**，区分活人与静置物品（书包、书本等）。

### 原理

```
人坐着：     ▁▃▅▂▄▆▅▃▂▁   ← 呼吸/微动 → 方差大
物品放着：   ▁▁▁▁▁▁▁▁▁▁▁   ← 完全静止 → 方差≈0
```

### 算法流程
1. 每轮循环将 6 路归一化压力值写入**环形缓冲区**（10 帧）
2. 对所有通道计算**样本方差**（无偏估计）
3. 取 6 通道平均方差
4. 若平均方差 < 0.5 **持续确认 5 次** → 判定为物品（防抖）
5. 一旦任一通道出现波动 → 瞬间恢复活人检测

### 行为变更
| 场景 | 以前 | 现在 |
|:-----|:-----|:-----|
| 椅子放书包 | 误判为端正坐姿，45分钟触发久坐提醒 | 识别为**物品**，不计时、不报警 |
| 人坐下不动 | 可能误判为物品（短暂过渡期） | 坐下瞬间有波动，仅5帧后稳定时才判物 |
| 人从静止到微动 | — | 方差立即 > 阈值，瞬间切回活人检测 |
| 拿走物品 | 压力归零→无人 | 压力归零→无人（拿走时有波动，不影响） |

### 配置参数
| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `OBJ_HISTORY_SIZE` | 10 | 历史窗口大小（~1秒数据） |
| `OBJ_VAR_THRESHOLD` | 0.5 | 方差阈值（归一化压力⁶²） |
| `OBJ_CONFIRM_CYCLES` | 5 | 连续确认次数（防抖动） |
