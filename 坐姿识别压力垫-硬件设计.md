# ⚙️ 坐姿识别压力垫 — 硬件设计

## 1. 电路原理

### FSR 分压电路
每个传感器连接为分压器配置：
```
5V ──┬── FSR ──┬── A0 (Arduino)
      │         │
      └── 10kΩ ──┘ GND
```

**ADC 读数估算：**
- 轻压(FSR≈10kΩ) → Vout≈2.5V → ADC≈512
- 重压(FSR≈1kΩ) → Vout≈4.5V → ADC≈922

### 完整接线表

| Arduino Nano Pin | 连接 |
|:----------------:|------|
| 5V | FSR 公共电源线 |
| GND | 下拉电阻公共地 |
| A0 | S1 - 前左 |
| A1 | S2 - 前右 |
| A2 | S3 - 中左 |
| A3 | S4 - 中右 |
| A4 | S5 - 后左 |
| A5 | S6 - 后右 |
| D13 (LED) | 坐姿指示 |

```

    ┌──────────────────────────────┐
    │  ┌──────┐           ┌──────┐ │
    │  │ S1   │  大腿/膝  │ S2   │ │  ← 距前方 5cm

    │      ┌──────┐ ┌──────┐       │
    │      │ S3   │ │ S4   │       │  ← 坐骨结节（关键）

    │      └──────┘ └──────┘       │
    │  ┌──────┐           ┌──────┐ │
    │  │ S5   │  臀部后方 │ S6   │ │  ← 距后方 5~10cm
    │  │ 后左 │           │ 后右 │ │
    │  └──────┘           └──────┘ │
    └──────────────────────────────┘
        椅子下方（靠背侧）
```

## 3. 防水封装方案

### 传感器级防水（6.2节）
```
FSR  →  ①焊点涂三防漆/热熔胶
      →  ②PVC热封袋密封（封口机140℃）+ 硅胶固定
      →  ③外裹0.1mm PU防水尼龙布
      →  ④排线出口热缩管密封
```
**⚠️ 禁忌：** 不要用环氧树脂直接灌封（改变压阻特性）；不要完全抽真空。

### PCB 级防水
- 控制盒与坐垫物理分离（ABS防水盒 IP65）
- PCB 喷涂三防漆（CRC 70）
- 进线口用防水格兰头（PG9）

## 4. 座椅加热与通风系统

### 架构
```
12V/2A 适配器
   │
   ├── LM2596 DC-DC → 5V → Arduino Nano
   │
   ├── IRLZ44N (D6 PWM) → 碳纤维加热垫 12V 15W
   │
   ├── IRLZ44N (D7 PWM) → 静音风扇 12V 0.1A
   │
   └── DHT22 (D8) → 温湿度传感器
```

### 自动控制逻辑
| 条件                   | 动作                |
| :------------------- | :---------------- |
| 温度 < 18°C            | 自动加热 50%          |
| 温度 > 25°C            | 关闭加热              |
| 温度 > 28°C 或 湿度 > 70% | 开启风扇              |
| 手动设定                 | 优先级最高，30min 后切回自动 |

**安全：** 30分钟自动关闭 + 65°C 温度保险丝 + 硬件看门狗

## 5. 透气结构

### 坐垫多层结构
```
① 3D透气网眼布（Coolmax）— 排汗速干
② 3mm透气毡（针刺棉）— 隔开传感器硬点
③ FSR传感器（嵌入海绵槽）
④ 20mm高弹海绵（Φ5mm打孔，间距15~20mm）
⑤ 5mm硬质海绵（支撑层）
⑥ 底部防滑布（点塑布）
```
