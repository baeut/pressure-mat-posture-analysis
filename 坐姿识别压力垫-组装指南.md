# 🔧 坐姿识别压力垫 — 完整组装施工指南

> 预计工时：**4~6 小时**（含等待硅胶固化时间）

---

## 工具准备

| 工具 | 用途 |
|:----|:-----|
| 电烙铁 + 焊锡 | 焊接电路 |
| 热风枪/家用封口机 | 热封袋封口（直发夹板≤150℃替代） |
| 空心冲子 Φ5mm | 海绵打孔 |
| 热熔胶枪 | 固定/密封 |
| 万用表 | 测试通断、电阻 |
| 剪刀、美工刀 | 裁剪 |

---

## 第一步：FSR 传感器防水封装（每只约10分钟）

1. 酒精棉擦拭 FSR 有源区，完全干燥
2. 焊接 15cm 硅胶线（红+黑），焊点涂热熔胶
3. 装入 PVC 热封袋，滴 3~5 滴硅胶固定传感器
4. 140℃ 热封口，排线口涂热熔胶 + Φ3mm 热缩管
5. 万用表测试：空载 >1MΩ，按压 <100kΩ

**完成6个后，用标签贴纸标记 S1~S6。**

---

## 第二步：坐垫透气结构制作

1. 20mm 高弹海绵打 Φ5mm 孔（梅花形排列，间距15~20mm）
2. **传感器正下方不打孔**（周围2cm区域保持平整）
3. 切割网眼布、透气毡、防滑布至 45×45cm
4. 按多层结构叠放：
   - 最下：防滑布
   - 5mm 硬质海绵
   - 20mm 打孔高弹海绵（嵌入传感器）
   - 3mm 透气毡
   - 最上：3D网眼布

---

## 第三步：电路焊接

1. 在洞洞板上焊接 6 组 10kΩ 下拉电阻
2. 焊接排针母座（供 Arduino Nano 插入）
3. 焊接排线端子（连接传感器）
4. 万用表通断测试

---

## 第四步：总装

1. 将密封好的传感器嵌入海绵对应位置
2. 传感器排线汇总到坐垫边缘出口
3. 安装控制盒（固定在椅子底部）
4. 插入 Arduino Nano，连接 USB 线
5. 上电测试

---

## 烧录与测试

```bash
# 编译
cd seat_sensor/
arduino-cli compile --fqbn arduino:avr:nano seat_sensor.ino

# 烧录（确认COM口）
arduino-cli upload -p COM3 --fqbn arduino:avr:nano seat_sensor.ino

# 启动上位机监测
python scripts/monitor.py --port COM3
```

**验证：** 分别按压 S1~S6，确认串口输出 JSON 中对应通道数值变化。
