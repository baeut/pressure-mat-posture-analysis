## 🧱 硬件方案

### 传感器 3×2 阵列布局
```
        椅子前方（膝盖侧）
    ┌─────────────────────┐
    │  S1(前左)    S2(前右) │  ← 大腿根部压力
    │                     │
    │  S3(中左)    S4(中右) │  ← 坐骨结节压力（关键）
    │                     │
    │  S5(后左)    S6(后右) │  ← 尾骨/臀部后方
    └─────────────────────┘
        椅子后方（靠背侧）
```

### 电路原理
每个 FSR 通过 **10kΩ 下拉电阻** 构成分压电路 → `analogRead` 值随压力变化。

## 💻 软件架构

```
Arduino Nano (seat_sensor.ino)
   ↓ USB串口 (115200, JSON格式)
上位机 (monitor.py / seat_bridge.py)
   ↓ 处理后分发
系统托盘弹窗 / 升降桌联动 / 微信推送 / Webhook
```

### 坐姿分类算法
中值滤波 → 归一化 → 压力重心(COG) → 阈值逻辑判断

| 坐姿 | 特征 |
|:----:|------|
| 🟢 端正 | 压力均匀分布 |
| 🟡 前倾 | S1+S2 >> S5+S6 |
| 🟡 后仰 | S5+S6 >> S1+S2 |
| 🟡 左倾 | 左侧总压力 > 右侧+阈值 |
| 🟡 右倾 | 右侧总压力 > 左侧+阈值 |
| 🔵 跷左腿 | S3 >> S4 且 S2 ≈ 0 |
| 🔵 跷右腿 | S4 >> S3 且 S1 ≈ 0 |
| ⚫ 无人 | 所有通道 < 阈值 |
